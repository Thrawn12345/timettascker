Phone Setup (Android)
Step 1 — Install Kiwi Browser (Play Store). It's the only Android browser that supports Chrome extensions.

Step 2 — Load the extension:
Open Kiwi → Menu → Extensions → Enable Developer Mode → Load Unpacked → select the extension/ folder (copy it to your phone first via USB or Google Drive).

Step 3 — Configure: Open the extension settings in Kiwi, set Device Name to something like Android Phone and Server URL to your Render cloud URL or your PC's LAN IP.

Manual app timer: Open android/tracker.html in any phone browser, add to Home Screen for quick access.
